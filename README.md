# Steam-Free
STEAMFREE — это настольное приложение на Python (PyQt6) для разблокировки игр Steam на стороне клиента двумя способами: SteamTools и GreenLuma. Программа загружает манифесты и ключи расшифровки из открытых репозиториев, подготавливает конфигурацию для выбранного инструмента и ведёт логи работы. Внутри есть система настроек.
